# EchoLearn AI 🧠
> An adaptive, inclusive classroom system powered by Gesture Recognition & Speech-to-Text..

## 🚀 Features
* **Live Class Mode:** Speech-to-Sign Language translation for Deaf accessibility.
* **Gesture Quiz:** Computer Vision-based answering system for Motor Impaired students.

## 🛠 Tech Stack
* Next.js / React
* MediaPipe (AI/ML)
* Web Speech API
* Tailwind CSS
